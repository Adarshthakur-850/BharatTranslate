BharatTranslate is a production-ready AI-powered Multilingual Translation System that supports 22 official Indian languages. It uses FastAPI for the backend, React+Tailwind for a high-performance frontend, and provides integration logic for Transformer-based models like `IndicTrans2` or `mBART`.

## Structure

- **frontend/**: React Vite Application built with TailwindCSS. Contains UI for Translation, History, and Settings.
- **backend/**: FastAPI application offering low-latency translation endpoints.
- **ml/**: Scaffolds for data processing, training pipelines, and inference evaluations.
- **deployment/**: Docker Compose, Dockerfiles, and Kubernetes manifests for container orchestration.
- **monitoring/**: Prometheus configurations for system health and load tracking.

## Getting Started

Check out `docs/setup_guide.md` for full installation instructions.

## Supported Languages

Hindi, Bengali, Tamil, Telugu, Marathi, Gujarati, Kannada, Malayalam, Punjabi, Odia, Assamese, Urdu, Sanskrit, Kashmiri, Konkani, Sindhi, Manipuri, Bodo, Dogri, Maithili, Santhali, Nepali, and English.
